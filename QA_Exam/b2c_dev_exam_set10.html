B2C Commerce Developer Exam - Practice Set 10

  1. A new custom controller was uploaded but returns 404. What is the most likely issue?

    Answer
    ✅ Correct: B
    Cartridge must be in the site path to resolve controllers.

  2. Where must cartridge code be uploaded for a specific code version?

    Answer
    ✅ Correct: B
    Deployed code for a specific version resides at /versioned/.../cartridges.

  3. You need to override app_storefront_base PDP template. Which two conditions are required?

    Answer
    ✅ Correct: A, C
    Same relative path/name and higher path precedence.

  4. Which is true about code versions?

    Answer
    ✅ Correct: B
    Sites independently reference a code version.

  5. After switching the referenced code version, the storefront still renders old content. Which two steps help?

    Answer
    ✅ Correct: A, B
    Clear application cache and invalidate template fragments.

  6. Why does cartridge order matter in SFRA?

    Answer
    ✅ Correct: B
    Left-to-right precedence controls resolution.

  7. Where is the active code version selected?

    Answer
    ✅ Correct: A
    Code Versions page manages the active version.

  8. What does dw.json configure?

    Answer
    ✅ Correct: B
    dw.json configures IDE/WebDAV connection and target version.

  9. A stakeholder requests different cartridge paths per locale within one site. Correct guidance?

    Answer
    ✅ Correct: B
    Path is site-level; handle locale differences in code/templates.

  10. Which artifact inside a cartridge defines cartridge metadata like id/description?

    Answer
    ✅ Correct: B
    app_*.properties holds cartridge metadata.

  11. You must enforce HTTPS on sensitive routes. What should you use?

    Answer
    ✅ Correct: A
    Use server.middleware.https.

  12. Two cartridges define the same controller route. Which implementation runs?

    Answer
    ✅ Correct: B
    The leftmost (earliest) cartridge in the path wins.

  13. Where can you import catalogs and inventory?

    Answer
    ✅ Correct: B
    Site Import & Export handles inbound data loads.

  14. To make a product attribute searchable, which two are required?

    Answer
    ✅ Correct: A, C
    Mark searchable and rebuild index.

  15. Where are storefront search refinements configured?

    Answer
    ✅ Correct: A
    Refinements are configured under Search.

  16. A one-way synonym “parka → winter coat” is required. Where is it configured?

    Answer
    ✅ Correct: B
    One-way synonyms live in Dictionaries.

  17. To show a new PDP attribute value, which two updates are typical?

    Answer
    ✅ Correct: A, B
    Extend model and update templates.

  18. Where are Logger categories and appenders configured?

    Answer
    ✅ Correct: A
    Configure logging under Admin > Operations.

  19. A marketer wants to assemble new landing pages with components. Which feature should be used?

    Answer
    ✅ Correct: A
    Page Designer supports componentized pages.

  20. After enabling a new locale, search returns fewer results. Which two are likely?

    Answer
    ✅ Correct: A, B
    Missing localized values or not reindexing reduces recall.

  21. To allow an integration to access a custom object via OCAPI Data, which two steps are required?

    Answer
    ✅ Correct: A, B
    ocapi.json permissions and client credentials are required.

  22. Production logging best practice:

    Answer
    ✅ Correct: A
    Use Logger with appropriate levels; limit DEBUG in prod.

  23. Where do you configure the site default currency and taxation type?

    Answer
    ✅ Correct: A
    Site Settings specify currency/taxation.

  24. To change availability label copy, which two places are typical?

    Answer
    ✅ Correct: A, B
    Properties + ISML typically control user-facing copy.

  25. Maintainable controller design includes:

    Answer
    ✅ Correct: A
    Keep controllers thin; push domain logic to helpers/decorators/factories.

  26. Which directive enables cacheable template fragments?

    Answer
    ✅ Correct: A
    iscache caches template fragments.

  27. Proper CSRF handling for forms requires:

    Answer
    ✅ Correct: A
    CSRF token generate/validate pattern.

  28. A route returns JSON. What two actions are correct?

    Answer
    ✅ Correct: A, B
    Correct content type and JSON body handling.

  29. Customize pricing behavior without editing base controllers using:

    Answer
    ✅ Correct: A
    Use hooks to modify pricing/tax behavior.

  30. A PLP performs synchronous HTTP per product tile. Which two help?

    Answer
    ✅ Correct: A, B
    Service Framework + batching/caching reduces per-item calls.

  31. Localize UI strings using:

    Answer
    ✅ Correct: A, B
    Use properties bundles and Resource helpers for i18n.

  32. A remote include shows stale promotion data. How to fix?

    Answer
    ✅ Correct: A
    Add vary keys for basket/promo/customer or disable include cache.

  33. Caching a personalized Shop API response safely requires:

    Answer
    ✅ Correct: A
    No cache or appropriate vary keys for personalization.

  34. A service returns HTTP 200 with an error payload. Correct handling is:

    Answer
    ✅ Correct: B
    Parse application-level status and respond accordingly.

  35. Storing PII from a form requires which two?

    Answer
    ✅ Correct: A, B
    Minimize/encrypt PII; enforce least-privilege access.

  36. A controller must stream a large CSV. Best approach:

    Answer
    ✅ Correct: A
    Stream CSV; avoid building large responses in memory.

  37. To allow baskets via OCAPI Shop, which two are required?

    Answer
    ✅ Correct: A, B
    OCAPI Shop configuration + client mapping.

  38. A nightly job imports inventory and reindexes search. Which two steps are typical?

    Answer
    ✅ Correct: A, B
    Import via script, then trigger indexing.

  39. Which inbound folders are commonly used for imports?

    Answer
    ✅ Correct: A, B
    impex src and src/instance are common inbound folders.

  40. When an import job intermittently fails due to file locks, what two mitigations help?

    Answer
    ✅ Correct: A, B
    Retries with backoff and schedule/lock coordination.

  41. Least-privilege OCAPI Data access for orders should include:

    Answer
    ✅ Correct: A, B
    Restrict attributes and use dedicated credentials.

  42. To rotate a service credential without downtime, which two steps help?

    Answer
    ✅ Correct: A, C
    Blue/green credentials or overlapping windows.

  43. A robust synchronous integration should include:

    Answer
    ✅ Correct: A, B
    Short timeouts, retries, and circuit breaker.

  44. Product deltas arrive hourly. Which indexing policy is most efficient?

    Answer
    ✅ Correct: A
    Use delta indexing for frequent updates.

  45. To write to a custom object via Data API you need:

    Answer
    ✅ Correct: A, B
    Write permissions/attributes and mapped client.

  46. Two job steps update the same records and collide. The best remedy is:

    Answer
    ✅ Correct: A
    Serialize or isolate windows to avoid conflicts.

  47. Prevent sensitive token leakage from services logs by:

    Answer
    ✅ Correct: A
    Redact tokens and avoid logging bodies unless masked.

  48. A Shop API endpoint varies by pricebook and locale. Safe caching strategy:

    Answer
    ✅ Correct: A
    No cache or vary keys (pricebook/locale).

  49. To reduce PDP TTFB, which two help?

    Answer
    ✅ Correct: A, B
    Precompute and cache static fragments.

  50. Profiler shows heavy per-tile service calls. Which two help?

    Answer
    ✅ Correct: A, B
    Batch/cache and precompute via jobs.

  51. Stale content persists after deployment. Which two are typical fixes?

    Answer
    ✅ Correct: A, B
    Clear app cache and invalidate iscache.

  52. A large JSON export OOMs. Correct fix is:

    Answer
    ✅ Correct: A
    Stream JSON to avoid OOM.

  53. Cache correctness issues on multi-locale/multi-currency sites often arise from:

    Answer
    ✅ Correct: A, B
    Missing vary keys/global cache leads to leakage/incorrectness.

  54. New locale shows English strings. Which two fixes apply?

    Answer
    ✅ Correct: A, B
    Add locale bundles and use Resource helpers.

  55. A route frequently times out on a downstream service. Which three techniques are best practice?

    Answer
    ✅ Correct: A, B, C
    Fail fast, retry, circuit breaker.

  56. After a release, PDP TTFB increased by 250ms. Which three diagnostics help?

    Answer
    ✅ Correct: A, B, C
    Profiling, targeted timers, and diffs isolate regressions.

  57. To reduce ISML template CPU time, which two help?

    Answer
    ✅ Correct: A, B
    Move logic out of templates and cache static parts.

  58. A PD component shows wrong currency when cached. Fix by:

    Answer
    ✅ Correct: A
    Vary by currency for correctness.

  59. Prevent duplicate POST submissions:

    Answer
    ✅ Correct: A, B
    Idempotency and client-side prevention.

  60. Catalog import and delta reindex sometimes overlap and fail. Best remedy:

    Answer
    ✅ Correct: A
