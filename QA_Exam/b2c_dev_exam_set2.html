B2C Commerce Developer Exam - Practice Set 2

  1. Which statement best describes the relationship between code versions and sites?

    Answer
    ✅ Correct: B
    Sites independently reference a code version in Settings.

  2. Which directory is used by WebDAV to upload a cartridge for the active code version?

    Answer
    ✅ Correct: A
    Code deploys under /versioned/cartridges/<code-version>/cartridges.

  3. In SFRA, where are HTTP routes for controllers typically defined?

    Answer
    ✅ Correct: B
    Routes are registered in server-side JS via server.get/post.

  4. When extending app_storefront_base with custom functionality, which path ordering is recommended?

    Answer
    ✅ Correct: B
    Custom first, base last to allow overrides.

  5. What is the purpose of “decorators” in SFRA?

    Answer
    ✅ Correct: B
    Decorators extend models cleanly.

  6. What is the primary benefit of the model-factory pattern in SFRA?

    Answer
    ✅ Correct: B
    Factories centralize construction and apply decorators.

  7. Which two files commonly appear in an SFRA controller module?

    Answer
    ✅ Correct: A, B
    Controllers and ISML are paired; ocapi/services files aren’t for routes.

  8. What is the impact of placing a custom cartridge after app_storefront_base in the site path?

    Answer
    ✅ Correct: B
    Base after custom means base can override later-resolved artifacts.

  9. Which action is required after uploading new controllers to the code version?

    Answer
    ✅ Correct: B
    Ensure cartridge path and clear caches when needed.

  10. Which statement best describes SFRA’s separation of concerns?

    Answer
    ✅ Correct: A
    Controllers thin; business logic in models/decorators; templates minimal.

  11. How do you verify that a new controller route is resolvable?

    Answer
    ✅ Correct: C
    Call the route directly to validate.

  12. What is the effect of clearing application cache in Business Manager?

    Answer
    ✅ Correct: B
    Cache clear refreshes rendered output.

  13. Where are product search synonyms configured?

    Answer
    ✅ Correct: A
    Synonyms live in Dictionaries.

  14. To make a custom product attribute searchable, which two actions are required?

    Answer
    ✅ Correct: A, C
    Mark searchable and reindex.

  15. Where do you configure payment processors and methods for a site?

    Answer
    ✅ Correct: B
    Payment config under Ordering.

  16. Which tool is used to import a catalog XML into a sandbox?

    Answer
    ✅ Correct: A
    Site Import & Export handles data loads.

  17. To display a new attribute on PDP, where might developers typically add code?

    Answer
    ✅ Correct: A, B
    Extend model and update templates to render.

  18. Which is the correct location to configure logs and custom categories?

    Answer
    ✅ Correct: B
    Logs configured in BM log settings.

  19. A merchandiser needs to curate content for the homepage. Which two features are most relevant?

    Answer
    ✅ Correct: A, B
    Content Assets/PD are merch tools.

  20. A data import fails with “invalid XML structure.” First step?

    Answer
    ✅ Correct: B
    Validate the input XML against schema.

  21. To expose a custom object externally via OCAPI Data, which two are needed?

    Answer
    ✅ Correct: A, B
    OCAPI permissions and client credentials.

  22. What is a common sign that you should introduce a custom object type?

    Answer
    ✅ Correct: B
    Use custom objects for domain data that doesn’t fit system objects.

  23. Where do you configure search refinements (facets)?

    Answer
    ✅ Correct: A
    Refinements (facets) configured under Search.

  24. After creating a new searchable attribute, the storefront still doesn’t return results. Which two steps are likely missing?

    Answer
    ✅ Correct: A, C
    Reindex; ensure data is populated and assigned.

  25. What middleware should wrap POST form routes to protect against CSRF?

    Answer
    ✅ Correct: B
    CSRF token generate/validate for POSTs.

  26. How do you send data from a controller to a template?

    Answer
    ✅ Correct: B
    res.render(template, data).

  27. A controller must return JSON. Which two steps are correct?

    Answer
    ✅ Correct: A, B
    Set JSON content-type and respond with JSON.

  28. Where should complex pricing logic be implemented for maintainability?

    Answer
    ✅ Correct: B
    Centralize in hooks/decorators for reuse.

  29. Which ISML tag enables a separately cachable component rendering?

    Answer
    ✅ Correct: B
    iscomponent supports separate caching units.

  30. Which three artifacts are typical when implementing a custom form?

    Answer
    ✅ Correct: A, B, D
    Form XML, CSRF-protected POST, and binding/validation.

  31. Which hook type allows you to modify basket or price calculation without editing controllers?

    Answer
    ✅ Correct: C
    Use pricing/tax calculation hooks.

  32. Best practice for logging messages from controllers/models?

    Answer
    ✅ Correct: B
    Use platform Logger with categories.

  33. ISML localization uses which mechanism?

    Answer
    ✅ Correct: B
    Properties files per locale.

  34. A controller uses multiple synchronous service calls per request leading to timeouts. Which two fixes help?

    Answer
    ✅ Correct: A, B
    Cache responses and add resiliency via Service Framework.

  35. A page fragment must vary by locale and currency. What is recommended?

    Answer
    ✅ Correct: B
    Use iscache/res.cachePeriod with vary keys.

  36. After deploying a new template, changes don’t appear. What are two likely remedies?

    Answer
    ✅ Correct: A, C
    Clear app cache or invalidate template cache.

  37. Which three Service Framework settings are essential for resilient integrations?

    Answer
    ✅ Correct: A, B, D
    Timeouts, resiliency, and parsing/caching.

  38. Where do you securely store API credentials for services?

    Answer
    ✅ Correct: A
    Store secrets in BM service credentials.

  39. To allow access to baskets via OCAPI Shop API, which two are required?

    Answer
    ✅ Correct: A, B
    OCAPI permissions and client config.

  40. A nightly job must import inventory and refresh search. Which two steps are typical?

    Answer
    ✅ Correct: A, C
    Import via script; delta search step.

  41. Which WebDAV directories are typically used for import sources?

    Answer
    ✅ Correct: A, B
    /impex/src[/instance] are import folders.

  42. A job intermittently fails due to locked resources. Which two mitigations help?

    Answer
    ✅ Correct: A, C
    Retries/backoff and pre-checks mitigate locks.

  43. When should an integration be asynchronous?

    Answer
    ✅ Correct: A, B
    Async when not needed synchronously; allows queuing/retry.

  44. To prevent sensitive data from appearing in logs, what is recommended?

    Answer
    ✅ Correct: B
    Redact before logging.

  45. OCAPI customization logic (hooks) lives where?

    Answer
    ✅ Correct: B
    Implement OCAPI hooks in a cartridge.

  46. A PIM pushes product deltas hourly. Efficient indexing strategy?

    Answer
    ✅ Correct: B
    Delta indexing for frequent updates.

  47. A service endpoint begins timing out. Which three actions improve resiliency?

    Answer
    ✅ Correct: A, B, C
    Fail fast, add retries/circuit break, and cache.

  48. Which two job step types are common in B2C Commerce?

    Answer
    ✅ Correct: A, B
    Script and System steps are standard.

  49. Which two platform-level caches are commonly used for storefront performance?

    Answer
    ✅ Correct: A, B
    iscache and response cache.

  50. Which two patterns reduce controller complexity?

    Answer
    ✅ Correct: A, C
    Keep controllers thin; push logic down; use services.

  51. For localization of UI text, which two practices are correct?

    Answer
    ✅ Correct: A, C
    Properties + Resource helpers.

  52. Which two anti-patterns hurt scalability?

    Answer
    ✅ Correct: A, B
    Repeated sync calls and heavy template logic are anti-patterns.

  53. A template snippet is static and identical for all users. Best caching approach?

    Answer
    ✅ Correct: B
    iscache (no vary) for static identical snippets.

  54. Which two steps help diagnose a slow route?

    Answer
    ✅ Correct: A, B
    Logs and profiler reveal hotspots.

  55. Which practice best protects against CSRF on forms?

    Answer
    ✅ Correct: B
    CSRF middleware + tokens.

  56. Where should secrets and API keys live?

    Answer
    ✅ Correct: A
    Store secrets in BM, not code.

  57. After deployment, part of the site still shows stale content. Which two actions are appropriate?

    Answer
    ✅ Correct: A, B
    Clear app/template caches.

  58. When should remote includes be preferred over local includes?

    Answer
    ✅ Correct: A
    Remote includes provide separate pipelines and cache behavior.

  59. Which two strategies reduce rendering time?

    Answer
    ✅ Correct: A, B
    Precompute and cache static parts.

  60. Which log-level policy is recommended for production?

    Answer
    ✅ Correct: B
    Use INFO for normal ops; escalate when needed; DEBUG sparingly.
