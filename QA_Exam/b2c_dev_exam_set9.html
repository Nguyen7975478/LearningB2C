B2C Commerce Developer Exam - Practice Set 9

  1. A custom cartridge was uploaded to the correct code version but not appearing in the site’s cartridge path selector. What’s the likely cause?

    Answer
    ✅ Correct: B
    A valid cartridge folder must contain a cartridge/ subfolder to be selectable.

  2. You need to override the base PDP template. Which two conditions must be true?

    Answer
    ✅ Correct: A, D
    Match relative path/name and ensure custom cartridge precedes base.

  3. Which path is used to deploy code via WebDAV for a specific code version?

    Answer
    ✅ Correct: B
    Deployed code lives at /versioned/cartridges/<code-version>/cartridges.

  4. Why does cartridge order matter?

    Answer
    ✅ Correct: B
    Path precedence is left‑to‑right; first match wins.

  5. After changing the site’s referenced code version, you still observe old templates. Which two actions are recommended first?

    Answer
    ✅ Correct: A, B
    Clear app cache and invalidate iscache to surface new templates.

  6. Where do you assign the site’s code version?

    Answer
    ✅ Correct: A
    Code Versions UI controls active version selection.

  7. What is dw.json primarily used for?

    Answer
    ✅ Correct: B
    dw.json configures IDE target and version for uploads.

  8. A team asks for different cartridge paths by locale on the same site. What is the proper guidance?

    Answer
    ✅ Correct: B
    Path is site‑level; handle locale variance in templates/models.

  9. You uploaded a valid cartridge but routes still 404. What’s the first thing to check?

    Answer
    ✅ Correct: B
    Ensure the cartridge is in the site path with correct ordering.

  10. Which file inside a cartridge typically contains cartridge metadata like id/description?

    Answer
    ✅ Correct: A
    app_*.properties contains cartridge metadata.

  11. To require HTTPS on certain routes, use:

    Answer
    ✅ Correct: A
    Use server.middleware.https on routes that require HTTPS.

  12. If two custom cartridges define the same controller route, which executes?

    Answer
    ✅ Correct: B
    The leftmost cartridge with that route handles it.

  13. Where do you import product/catalog/inventory data?

    Answer
    ✅ Correct: A
    Use Site Import & Export for inbound data files.

  14. To make a new product attribute searchable, which two steps are required?

    Answer
    ✅ Correct: A, C
    Mark searchable and rebuild index to take effect.

  15. Where are storefront search refinements (facets) configured?

    Answer
    ✅ Correct: A
    Refinements are configured under Search > Search Refinements.

  16. A one‑way synonym is needed (“sneakers” → “running shoes,” but not vice versa). Where do you configure it?

    Answer
    ✅ Correct: A
    Dictionaries support one‑way synonyms.

  17. You need to show a new PDP attribute value. Which two layers are commonly updated?

    Answer
    ✅ Correct: A, B
    Extend model and update templates.

  18. Where do you configure log categories and appenders?

    Answer
    ✅ Correct: A
    Logging config is managed under Admin > Operations.

  19. A marketer wants to assemble landing pages from reusable components. Which feature is most appropriate?

    Answer
    ✅ Correct: A
    Page Designer enables componentized page assembly.

  20. Search results decreased after enabling a new locale. Which two are likely causes?

    Answer
    ✅ Correct: A, B
    Missing locale attributes or not reindexing reduces results.

  21. You must expose a custom object through OCAPI Data. Which two steps are required?

    Answer
    ✅ Correct: A, B
    ocapi.json permissions and client credentials are both required.

  22. Logging best practice in production is:

    Answer
    ✅ Correct: A
    Use Logger with proper levels; keep DEBUG targeted.

  23. Where do you change the site’s default currency and taxation settings?

    Answer
    ✅ Correct: A
    Site Settings manage currency/taxation.

  24. To change copy for availability labels on PLP/PDP, which two places are typical?

    Answer
    ✅ Correct: A, B
    Properties and ISML typically drive label text.

  25. Which approach keeps controllers maintainable?

    Answer
    ✅ Correct: A
    Thin controllers; logic in helpers/decorators/factories.

  26. Which directive enables cacheable template fragments?

    Answer
    ✅ Correct: A
    iscache supports fragment caching in ISML.

  27. Proper CSRF handling for forms is:

    Answer
    ✅ Correct: A
    CSRF token pattern (generate on GET; validate on POST).

  28. A route returns JSON. Which two are correct?

    Answer
    ✅ Correct: A, B
    Serve JSON with proper content type and JSON body.

  29. You need to customize pricing without editing base controllers. What should you use?

    Answer
    ✅ Correct: A
    Use hooks to introduce/override calculation logic.

  30. A PLP calls a synchronous service per item. Which two improvements help?

    Answer
    ✅ Correct: A, B
    Service Framework and batching/caching remove N+1 patterns.

  31. To localize text, use:

    Answer
    ✅ Correct: A, B
    Properties bundles and Resource helpers provide i18n.

  32. A remote include shows stale promo content. What is the likely fix?

    Answer
    ✅ Correct: A
    Add vary for basket/promo/customer context or disable caching.

  33. Caching a personalized Shop API response safely requires:

    Answer
    ✅ Correct: A
    No cache or include appropriate vary keys for personalization.

  34. Your service sometimes returns HTTP 200 with an application error in the body. Proper handling is:

    Answer
    ✅ Correct: B
    Inspect application‑level status in the response to detect errors.

  35. For handling PII in form submissions, which two are correct?

    Answer
    ✅ Correct: A, B
    Limit and secure PII; least‑privilege OCAPI access.

  36. A controller must output a large CSV. Best approach is:

    Answer
    ✅ Correct: A
    Streaming avoids excessive memory usage.

  37. To allow baskets via OCAPI Shop, you must:

    Answer
    ✅ Correct: A, B
    Configure ocapi.json Shop + map client credentials.

  38. A nightly job imports inventory and reindexes search. Which two steps are typical?

    Answer
    ✅ Correct: A, B
    Import via script; trigger indexing via system step.

  39. Which inbound folders are commonly used for imports?

    Answer
    ✅ Correct: A, B
    impex src and src/instance are used for inbound files.

  40. Two jobs sometimes collide updating the same records. Which approach helps?

    Answer
    ✅ Correct: A, B
    Locking/serialization and retries mitigate collisions.

  41. Least‑privilege OCAPI Data access for orders should include:

    Answer
    ✅ Correct: A, B
    Limit attributes, and use dedicated credentials.

  42. To rotate a service credential without downtime, which two steps help?

    Answer
    ✅ Correct: A, C
    Blue/green credentials or overlapping key windows.

  43. A robust synchronous integration should include:

    Answer
    ✅ Correct: A, B
    Fail fast, retry transients, and protect with a circuit breaker.

  44. Product deltas arrive hourly. Which indexing policy is most efficient?

    Answer
    ✅ Correct: A
    Delta reindexing after imports is efficient.

  45. To write to a custom object via Data API, you need:

    Answer
    ✅ Correct: A, B
    Write permissions and client mapping are required.

  46. Two job steps update the same object set and collide. What’s the remedy?

    Answer
    ✅ Correct: A
    Serialize access or separate windows to avoid conflicts.

  47. Prevent sensitive token leakage from service logs by:

    Answer
    ✅ Correct: A
    Redact before logging; avoid logging sensitive bodies.

  48. A Shop API endpoint varies by pricebook and locale. Safe caching strategy is:

    Answer
    ✅ Correct: A
    No cache or use vary by price group/locale.

  49. To reduce PDP time‑to‑first‑byte, which two are recommended?

    Answer
    ✅ Correct: A, B
    Precompute and cache static fragments.

  50. Profiler shows heavy spend on per‑tile service calls. Which two mitigations help most?

    Answer
    ✅ Correct: A, B
    Batch/cache or precompute via jobs.

  51. After deploy, a page remains stale. Which two actions are typical?

    Answer
    ✅ Correct: A, B
    Clear application cache and template cache.

  52. A large JSON export endpoint OOMs. Proper approach is:

    Answer
    ✅ Correct: A
    Stream JSON to avoid OOMs.

  53. Cache correctness issues on multilingual/multicurrency sites often stem from:

    Answer
    ✅ Correct: A, B
    Missing varies or global cache lead to leakage/incorrectness.

  54. New locale shows English strings. Which two fixes are standard?

    Answer
    ✅ Correct: A, B
    Add locale bundles and use Resource helpers.

  55. A route frequently times out on a downstream service. Which three resilience techniques are best practice?

    Answer
    ✅ Correct: A, B, C
    Short timeouts, retries with backoff, and circuit breaker.

  56. After a release, PDP TTFB increased by 220ms. Which three diagnostics help?

    Answer
    ✅ Correct: A, B, C
    Profiling, targeted timers, and code diffs isolate regressions.

  57. To reduce ISML template CPU time, which two are recommended?

    Answer
    ✅ Correct: A, B
    Move logic out of templates and cache fragments.

  58. A Page Designer component shows wrong currency when cached. What should you do?

    Answer
    ✅ Correct: A
    Add vary‑by‑currency where cached.

  59. To prevent duplicate POST effects (e.g., double‑click), which two mitigations help?

    Answer
    ✅ Correct: A, B
    Idempotency and UI prevention stop duplicates.

  60. Catalog import and delta reindex sometimes overlap and fail. What’s the best remedy?

    Answer
    ✅ Correct: A
    Coordinate and lock; use retries/backoff for robustness.
