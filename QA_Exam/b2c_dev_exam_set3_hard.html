B2C Commerce Developer Exam - Practice Set 3 (Hard)

  1. You deploy two custom cartridges that both define the same controller route path. Cartridge order is: app_custom_extA, app_custom_extB, app_storefront_base. Which route executes when the storefront requests that path?

    Answer
    ✅ Correct: C
    Cartridge path is resolved left-to-right; first match wins (custom extA).

  2. A site references code version “23.10.1” in Settings. You deploy new code to “23.12.0” and change the site’s referenced version, but overrides don’t appear. Which two actions are most likely required?

    Answer
    ✅ Correct: A, C
    Clear caches; ensure the new version’s cartridges are referenced in the site path.

  3. You need to validate which cartridge in the path is resolving a given ISML template. What’s the most precise approach?

    Answer
    ✅ Correct: B
    Tag each template candidate with a unique marker to observe which renders.

  4. In SFRA, which statement about middleware ordering is most accurate?

    Answer
    ✅ Correct: B
    Middleware runs in registered order; earlier middleware can short-circuit.

  5. You added a decorator that sets product.custom.badge = 'VIP' during model build. On PDP, some products render the badge, others don’t, though all have the attribute value. Which two causes are likely?

    Answer
    ✅ Correct: A, B
    Caching without proper vary keys can surface stale/other-product data.

  6. You extend app_storefront_base Product model via a factory that chains base and custom decorators. A teammate moves the custom decorator to run before the base builder. What subtle issue might occur?

    Answer
    ✅ Correct: B
    Running custom decorator before base can access missing fields; order matters.

  7. A controller uses a remote include to render a cart mini-summary. Occasionally, users see stale totals after promotions change. Which two misconfigurations can cause this?

    Answer
    ✅ Correct: A, B
    Missing vary on remote include or parent response cache causes stale totals.

  8. A site defines a cartridge path at the Organization level and another at the Site level. Which statement is correct?

    Answer
    ✅ Correct: B
    Paths merge; Site-level ordering takes precedence.

  9. You introduce a new code version and keep the same cartridge path. A single custom script resolves differently than before. Which is the best first step?

    Answer
    ✅ Correct: B
    Verify identical structure and presence of the script in the target version.

  10. In SFRA, which is the most accurate description of the role of controllers vs. helpers/decorators?

    Answer
    ✅ Correct: B
    Keep controllers thin; push business logic to helpers/decorators.

  11. Your custom storefront experience adds a controller that returns JSON for client-side rendering. It is cached at the CDN with no vary keys. Which two risks are introduced?

    Answer
    ✅ Correct: A, C
    Cached JSON without vary risks data leakage and stale personalization.

  12. After adding a cartridge with a higher path priority, a few Page Designer components stop rendering. What is the MOST likely reason?

    Answer
    ✅ Correct: A
    Higher-priority cartridge may shadow PD component templates/scripts.

  13. A new searchable product attribute is defined and populated. The attribute is still not searchable after a reindex. Which two issues are likely?

    Answer
    ✅ Correct: A, B
    Non-indexable type or attribute not attached to product type block search.

  14. Which approach yields the best recall for misspellings on popular search terms without over-matching?

    Answer
    ✅ Correct: B
    Use curated synonyms + tuned fuzzy thresholds to balance recall/precision.

  15. You add a refinement definition but it does not appear on category pages. Which two steps are commonly missed?

    Answer
    ✅ Correct: A, B
    Assign to menu and reindex so values are available.

  16. A merchandiser wants “color family” refinements grouped from granular colors (e.g., “navy,” “sky”) without changing source attributes. Which is the best approach?

    Answer
    ✅ Correct: B
    Use mapping rules to bucket values into a refinement dimension.

  17. Delta indexing is configured but new products aren’t appearing promptly. Which two misconfigurations commonly cause this?

    Answer
    ✅ Correct: A, B
    Missing delta trigger or attribute not searchable prevents prompt appearance.

  18. You need to expose a variant-level custom attribute in search facets. Which statement is accurate?

    Answer
    ✅ Correct: B
    Variant attributes can be indexed and mapped to facets with proper config.

  19. A guided search page needs a custom sort by “margin” (computed server-side). Which two are best practices?

    Answer
    ✅ Correct: A, C
    Precompute margin and define a sort using that field.

  20. A site has multi-locale catalogs. Searches in one locale return far fewer results than another. What are two likely causes?

    Answer
    ✅ Correct: A, C
    Missing localized values or tokenizer differences reduce results.

  21. Your relevance degraded after turning on aggressive stemming. What is the safest rollback?

    Answer
    ✅ Correct: B
    Taper stemming and validate via A/B and logs.

  22. You introduced a “brand” synonym expansion that maps many unrelated terms. Which remediation is best?

    Answer
    ✅ Correct: A
    One-way, precise expansions prevent over-broad matching.

  23. You must make a new attribute searchable, sortable, and refinable. Which two steps are essential?

    Answer
    ✅ Correct: A, B
    Enable attribute flags and include it in refinements/sort config.

  24. A custom index field inflates index size, slowing queries. Which optimizations help most?

    Answer
    ✅ Correct: A, B
    Reduce precision and store normalized tokens to shrink the index.

  25. A POST endpoint accepts JSON payloads and updates a custom object. Which two platform precautions are mandatory?

    Answer
    ✅ Correct: A, B
    Protect POSTs and validate inputs/size to prevent abuse.

  26. Your service call must fail fast during peak traffic to protect storefront performance. Which three configurations should you tune?

    Answer
    ✅ Correct: A, B, C
    Timeouts, CB, and backoff retries protect storefront under stress.

  27. You add a new Shop API OCAPI GET that is personalized. Which two requirements are critical for caching correctness?

    Answer
    ✅ Correct: B
    For personalized endpoints, either avoid caching or vary appropriately.

  28. A teammate added pricing logic directly in the PDP controller that also returns JSON for headless clients. What’s the MAIN maintainability risk?

    Answer
    ✅ Correct: B
    Logic diverges from hooks; duplication across outputs becomes fragile.

  29. Your basket calculation relies on a hook that sometimes runs after totals are displayed. Which is the best fix?

    Answer
    ✅ Correct: B
    Ensure hook order executes before totals; deterministic placement.

  30. A form persists PII to a custom object. Which two practices are mandatory?

    Answer
    ✅ Correct: B, C
    Minimize/secure PII; lock down OCAPI permissions.

  31. Your service occasionally returns a 200 with an error payload. How do you robustly detect failure?

    Answer
    ✅ Correct: B
    Parse payload-level status and schema to detect logical failures.

  32. An OCAPI Data POST intermittently fails with 403 for a client. Which two root causes are likely?

    Answer
    ✅ Correct: A, B
    403 often indicates auth or ocapi.json attribute permission issues.

  33. A hook implementation requires sharing context between pre- and post-steps. Which technique is appropriate?

    Answer
    ✅ Correct: B
    Use request-scoped context; avoid globals and persistent writes.

  34. Your checkout has a custom POST that sometimes receives duplicate submissions (e.g., double-click). Which two mitigations help?

    Answer
    ✅ Correct: A, B
    Idempotency and UI disable prevent duplicate effects.

  35. A controller needs to stream a large JSON export to the browser without exhausting memory. Which approach is best?

    Answer
    ✅ Correct: B
    Stream chunks to the response; avoid building huge in-memory JSON.

  36. Your custom object update job fails when running concurrently with another job that touches the same records. Best practice?

    Answer
    ✅ Correct: B
    Coordinate schedules/locks and implement backoff.

  37. Which two statements reflect least-privilege OCAPI configuration?

    Answer
    ✅ Correct: A, C
    Least privilege and separate credentials are core; never hardcode secrets.

  38. A new external client requires access to order data via OCAPI Data. Which three steps are essential?

    Answer
    ✅ Correct: A, B, D
    Credentials, ocapi.json, and PII minimization.

  39. Which two practices help avoid sensitive data in logs?

    Answer
    ✅ Correct: A, D
    Redact/mask and avoid logging bodies unless essential.

  40. Your hardening review flags that CSRF tokens sometimes aren’t validated on POST. What is the correct remediation?

    Answer
    ✅ Correct: B
    Generate token on GET, validate on POST for CSRF protection.

  41. An auditor asks how you segregate duties between environments (dev/test/prod). Which two controls are relevant?

    Answer
    ✅ Correct: A, C
    Distinct creds/roles per environment; controlled replication.

  42. You must rotate a service credential without downtime. Which two steps help?

    Answer
    ✅ Correct: A, D
    Blue/green credentials or dual-key transition when supported.

  43. A Shop API endpoint exposes personalized recommendations. What’s the safest caching strategy?

    Answer
    ✅ Correct: B
    No cache or vary by customer/locale/currency for personalization.

  44. A partner integration requests broad OCAPI write on system objects. What is the recommended response?

    Answer
    ✅ Correct: B
    Grant only what’s needed, scoped and time-bound.

  45. Your logs show frequent 401s on a service call from production only. Which two root causes are most likely?

    Answer
    ✅ Correct: A, B
    Wrong creds or IP allow-lists often cause 401 in prod.

  46. A new security policy mandates masking card PANs end-to-end. Which two platform strategies are applicable?

    Answer
    ✅ Correct: A, B
    Tokenize; redact/mask; never store full PANs.

  47. Your SiteGenesis legacy code uses XML pipelines and you must migrate a critical flow to SFRA. What’s the safest path?

    Answer
    ✅ Correct: B
    Incremental migration with isolation and tests is safest.

  48. An OCAPI customization must run validation before writing to a system object. Where does the logic live?

    Answer
    ✅ Correct: B
    OCAPI validation logic is in hook implementations in a cartridge.

  49. A route renders a product grid and is slow under load. Which three improvements are typical?

    Answer
    ✅ Correct: A, B, D
    Response caching, precompute, and batched/cached lookups help.

  50. You discover double JSON serialization (stringified strings). What is the clean fix?

    Answer
    ✅ Correct: B
    Avoid double-stringify; let res.json serialize once.

  51. Profiler shows 40% of time in a synchronous service call on each tile of a product list. Which two remedies help?

    Answer
    ✅ Correct: A, C
    Batch/cache or precompute via jobs; remove per-item calls.

  52. Stale template content persists despite cache clears. Which two lesser-known causes exist?

    Answer
    ✅ Correct: A, B
    Remote include cache and CDN can serve stale content independently.

  53. A job fails only when catalog import and search delta reindex overlap. What’s the best mitigation?

    Answer
    ✅ Correct: B
    Coordinate schedules and locking to avoid overlaps.

  54. You need to diagnose a memory spike during peak hours. Which two tools/approaches are relevant?

    Answer
    ✅ Correct: A, C
    Use profiler/log correlation; platform doesn’t provide heap dumps.

  55. The site occasionally returns 503 from a downstream service chain. Which three resilience techniques are most effective?

    Answer
    ✅ Correct: A, B, C
    CB, retries with backoff, and graceful fallbacks; never retry in ISML.

  56. A controller endpoint must stream a CSV export of orders for one tenant while preventing cross-tenant leaks. Which two safeguards are required?

    Answer
    ✅ Correct: A, C
    Scope/authorize per tenant and stream CSV with correct headers.

  57. A Page Designer component shows wrong currency on cache hits. Which root cause is most likely?

    Answer
    ✅ Correct: A
    Add vary-by-currency for correct cache partitioning.

  58. An integration needs at-least-once delivery semantics for outbound events. Which approach is best?

    Answer
    ✅ Correct: B
    Queue with IDs and dedupe for at-least-once semantics.

  59. The storefront performs excessive DDoS-like traffic to your service due to a client bug. Which two mitigations help quickly?

    Answer
    ✅ Correct: A, C
    Rate-limit and cache negatives to absorb bursts.

  60. A regression reduced PDP TTFB by 300ms. You must isolate the cause. Which three steps are effective?

    Answer
    ✅ Correct: A, B, D
    Profiling deltas, code diff, and targeted timings isolate regressions.
