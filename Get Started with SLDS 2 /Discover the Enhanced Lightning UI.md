## Introduction to Enhanced Lightning UI (Cosmos)
- The Enhanced Lightning UI (also called the **Cosmos** theme) is Salesforce’s next-generation user interface for better usability, accessibility, and a modern look and feel across the platform.
- Cosmos incorporates design updates powered by Salesforce Lightning Design System 2 (SLDS 2), focusing on clean aesthetics, flexible layouts, and responsive behavior.
- The new UI rolls out to Salesforce products starting Spring/Summer ‘25. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/enhanced-lightning-ui-quick-look/discover-enhanced-lightning-ui?trail_id=get-started-with-slds-2))

---

## New User Experience: Highlights & Improvements
Key areas where users will see improvements:
- **Modernized Visuals**: contemporary color palettes, friendlier fonts, increased whitespace, rounded corners, softer styling, fresh icons, better contrast, and dark mode support.
- **Enhanced Usability**: clearer navigation and grouping, more context-aware actions, easier on the eyes, and optimized workflow for all users.
- **Faster Performance**: optimized front-end, smaller, more flexible components for quicker load and interaction times.
- **Accessibility**: meets and exceeds latest accessibility standards, better support for assistive tech, contrast for low-vision, and keyboard navigation improvements.
- **Consistency**: unified experience across Salesforce products and new Lightning Apps. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/enhanced-lightning-ui-quick-look/discover-enhanced-lightning-ui?trail_id=get-started-with-slds-2))

---

## Who Benefits From Cosmos
- **Admins**: easier to configure and guide users with consistent, branded theming.
- **Developers**: easier to build apps/components that match Salesforce styling without custom complexity.
- **End Users**: enjoy a modern, organized, accessible user interface for daily CRM and productivity work.
- **Organizations**: present a unified, “future-ready” look for customers and partners as Salesforce iterates new features. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/enhanced-lightning-ui-quick-look/discover-enhanced-lightning-ui?trail_id=get-started-with-slds-2))

### Giới thiệu Cosmos/Enhanced Lightning UI
- Enhanced Lightning UI (chủ đề **Cosmos**) là UI thế hệ mới của Salesforce: tập trung vào usability, accessibility, và giao diện hiện đại nhất quán toàn platform.
- Áp dụng các update từ SLDS 2: layout linh hoạt, responsive, thẩm mỹ tinh tế.
- Rollout từng phần tại Spring/Summer ‘25. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/enhanced-lightning-ui-quick-look/discover-enhanced-lightning-ui?trail_id=get-started-with-slds-2))

---

### Cải tiến nổi bật
- Visual hiện đại: màu sắc mới, font thân thiện, nhiều white space, bo viền, biểu tượng mới, dark mode.
- Usability tốt hơn: navigation rõ ràng, action tiện dụng, workflow tối ưu.
- Nhanh hơn: component nhẹ, load nhanh hơn.
- Accessible: chuẩn accessibility mới, chất lượng cao cho người dùng trợ năng, tab dễ điều hướng.
- Consistency: trải nghiệm đồng bộ trên mọi sản phẩm Salesforce. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/enhanced-lightning-ui-quick-look/discover-enhanced-lightning-ui?trail_id=get-started-with-slds-2))

### Ai hưởng lợi?

- **Tổ chức**: look & feel đồng bộ khi tích hợp features mới. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/enhanced-lightning-ui-quick-look/discover-enhanced-lightning-ui?trail_id=get-started-with-slds-2))
