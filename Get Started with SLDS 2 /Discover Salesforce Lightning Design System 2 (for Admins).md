## SLDS 2 and the Cosmos Theme
- **SLDS 2** is the newest version of the Salesforce Lightning Design System underpinning all Lightning UI going forward.
- The **Cosmos** theme (default for SLDS 2) brings a modern, branded look-and-feel.
- Cosmos is centrally managed, aligns with accessibility/consistency standards, and is activated in Setup under Themes & Branding.
- The theme supports new dark mode, improved tokens, and SLDS 2-powered customization.
- All future Salesforce UI enhancements are based on SLDS 2/Cosmos, making this mandatory for organizations wanting the latest design features. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/salesforce-lightning-design-system-2-for-admins/discover-salesforce-lightning-design-system-2-admin?trail_id=get-started-with-slds-2))

## Benefits and Key Concepts for Admins
- **Easy brand alignment**: set brand colors, logos, and theming centrally.
- **Automatic updates**: new styles, UI features, and accessibility improvements arrive with each release.
- **Accessible by default**: conformance to WCAG accessibility standards.
- **Seamless consistency**: custom Lightning Apps and components inherit the theme—less need to override styling.
- **Faster configuration**: admins use drag-and-drop tools in Theme editor (no-code) to align apps to org branding.
- **Immediate benefit after enablement**: activating Cosmos/SLDS 2 refreshes the org’s look for all users at once.
  ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/salesforce-lightning-design-system-2-for-admins/discover-salesforce-lightning-design-system-2-admin?trail_id=get-started-with-slds-2))

## Comparison: SLDS 1 vs SLDS 2 for Admins
SLDS 1:
- Fewer theming/customization options
- Less unified experience (patchwork styles in large orgs)
- Fewer accessibility features
- Outdated look for modern users

SLDS 2 (Cosmos):
- Managed in Setup (admin-friendly)
- Consistent, modern UI
- Aligns with user, customer, and brand expectations
- Future proof: get new design capabilities automatically

### SLDS 2 + Cosmos Theme
- SLDS 2 là nền tảng thiết kế mới (tùy chỉnh cao, modern, accessible, nhất quán) toàn Salesforce.
- Theme Cosmos mặc định của SLDS 2 đem lại look “branded” và activate trong Setup > Themes & Branding.
- Hỗ trợ dark mode, tokens mới, tất cả đổi mới UI tương lai đều chạy trên SLDS 2. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/salesforce-lightning-design-system-2-for-admins/discover-salesforce-lightning-design-system-2-admin?trail_id=get-started-with-slds-2))

---

### Lợi ích cho admin
- Branding dễ, update UI tự động, theme accessible mặc định.
- Custom Lightning App chịu theme, ít code override.
- Theme editor kéo-thả nhanh, trải nghiệm đồng bộ ngay sau bật Cosmos cho toàn org.

---

### So sánh SLDS 1 vs SLDS 2
- SLDS 1: ít chọn theme, style rối, ít accessibility, nhìn cũ.
- SLDS 2: setup admin, UI hiện đại/đồng nhất, đáp ứng nhu cầu brand/customer/user, tự động nhận tính năng mới.
