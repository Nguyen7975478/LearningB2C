## English

## What categories do (beyond hierarchy)
- Categories provide more than a hierarchical structure:
    - Category attribute settings control storefront appearance.
    - Categories can be configured with scheduling, linking, search refinements, and results sorting.

## Create a category (storefront catalog) — unit walkthrough
1) Business Manager → Merchant Tools | Site | Products and Catalogs | **Catalogs**
2) Select catalog: `cloud-kicks-usa`
3) Click **New**
4) Create the root category (does not appear on storefront):
    - Category ID: `cloud-kicks-category`
    - Name: `Cloud Kicks`
    - Apply
5) Create a category under the root:
    - Back → click Cloud Kicks category link
    - New
    - Category ID: `sports-equipment`
    - Name: `Sports Equipment`
    - Apply
6) Create a subcategory (unit continues with subcategory creation following the same New-under-parent pattern)

## Vietnamese

## Categories dùng để làm gì (không chỉ là hierarchy)
- Categories không chỉ tạo cấu trúc phân cấp:
    - Category attribute settings điều khiển hiển thị trên storefront.
    - Có thể cấu hình scheduling, linking, search refinements, results sorting.

## Tạo category trong storefront catalog (walkthrough)
1) Merchant Tools | Site | Products and Catalogs | **Catalogs**
2) Chọn catalog `cloud-kicks-usa`
3) **New**
4) Tạo root category (không hiển thị trên storefront):
    - ID `cloud-kicks-category`
    - Name `Cloud Kicks`
    - Apply
5) Tạo category con:
    - Back → click Cloud Kicks link
    - New
    - ID `sports-equipment`
    - Name `Sports Equipment`
    - Apply
6) Tạo subcategory (tiếp tục theo cách tạo New dưới parent)
