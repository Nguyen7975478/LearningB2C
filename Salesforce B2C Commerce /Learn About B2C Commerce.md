- The site shows both items are available at a local store, and he arranges **in-store pickup** on the way out of town. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/cc-digital/learn-about-shopping-experience))

## Learn About Agentforce Commerce for B2C
- A team of Salesforce **admins, merchandisers, and developers** uses Agentforce Commerce for B2C to deliver the experience.
- Agentforce Commerce for B2C is the ecommerce solution to deliver scalable, feature-rich, fully localized storefronts.
- It integrates with other Agentforce Commerce products such as:
    - Order Management (OM)
    - Salesforce Payments
    - Retail Cloud POS (POS)
- This unified integration helps personalize online + in-store experiences to increase loyalty, grow revenue, and lower cost of ownership. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/cc-digital/learn-about-shopping-experience))

## Build Your Storefront
The unit states there are **three main options**:

1) **Storefront Reference Architecture (SFRA)** (server-side framework for fast launch)
- Accelerated development with preconfigured framework, reusable components, templates.
- Customization options; access to APIs including Agentforce Commerce for B2C API and Script API.
- Supports AI-powered personalization (active merchandising, Einstein recommendations).
- Easy content management (Page Designer, content assets) so merchants can update content without deep technical skill.
- Performance and scalability for high traffic/complex operations.
- Choose SFRA for fast launch with a powerful server-side framework and direct integration path / compatibility needs. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/cc-digital/learn-about-shopping-experience))

2) **Composable Storefront + Progressive Web App (PWA) Kit** (API-first headless)
- Headless architecture decouples front end and back end (flexible UX, easier third-party connections, parallel FE/BE dev).
- Prebuilt tools/templates; Salesforce-managed runtime optimized for performance and scalability.
- Better mobile performance; modular development for scalability; simplified maintenance; future-ready tech.
- Choose when you want maximum flexibility/growth and can invest in front-end + API integration skills. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/cc-digital/learn-about-shopping-experience))

3) **Bring-Your-Own Storefront (headless)**
- Use your own front-end tech, connect to Agentforce Commerce for B2C via APIs.
- Advantages listed: flexibility/UX customization, third-party integration, future-proofing, omni-channel reach, consistent experience, optimized performance/SEO, developer empowerment.
- Suitable for advanced teams with requirements beyond prebuilt frameworks. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/cc-digital/learn-about-shopping-experience))

### Salesforce Commerce APIs (SCAPI) for headless options
- SCAPI enables decoupled experiences and supports:
    - extending existing code investments,
    - creating fast unique experiences to optimize cart conversions,
    - integrating third-party systems/apps,
    - customizing B2C platform implementations. ([trailhead.salesforce.com](https://trailhead.salesforce.com/content/learn/modules/cc-digital/learn-about-shopping-experience))
